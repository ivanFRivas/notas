#git #branch

cada rama tiene su nombre.

Renombrar una rama: git branch -m nuevo-nombre-de-la-rama

Listar y explorar ramas

git branch: Lista tus ramas locales. La rama actual aparecerá marcada con un asterisco y resaltada.
git branch -a: Muestra todas las ramas (tanto locales como las de seguimiento remoto).
git branch -v: Lista las ramas locales junto con el hash del último commit y su mensaje.

Creación y navegación

git branch [nombre-de-la-rama]: Crea una nueva rama basada en tu commit actual.
git switch [nombre-de-la-rama]: Cambia a la rama especificada (para versiones modernas de Git).
git switch -c [nombre-de-la-rama]: Crea una rama y se mueve a ella inmediatamente en un solo paso.

Eliminar y renombrar

git branch -d [nombre-de-la-rama]: Elimina de forma segura la rama seleccionada (solo funciona si ya fue fusionada).
git branch -D [nombre-de-la-rama]: Fuerza la eliminación de la rama, incluso si tiene cambios sin fusionar.
git branch -m [nuevo-nombre]: Renombra la rama en la que te encuentras actualmente

Configuración de seguimiento remoto

git branch -u origin/[nombre-de-la-rama]: Vincula tu rama local actual con la rama remota correspondiente.
git branch -vv: Muestra las ramas locales y te indica si están sincronizadas, adelantadas o atrasadas respecto al servidor remoto.
